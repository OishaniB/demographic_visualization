**Demographic Visualization**



This project analyzes a demographic dataset to explore patterns in age, gender, marital status, income, car ownership, and education using R.
The goal is to uncover insights about how income and car prices relate to other demographic factors.



**#Key Features**

\- Data import and cleaning

\- Exploratory Data Analysis (EDA) on demographic variables

\- Visualizations:

  - Age and income distributions

  - Gender distribution

&nbsp; - Car price category insights

&nbsp; - Car price by gender

  - Income by education level

&nbsp; - Income vs car price



**#Dataset**

Contains demographic information like age, gender, marital status, income, income category, car price, car price category and education.



**#How to Run**

1\. Open the R script in RStudio.

2\. Make sure the dataset CSV file is in the same folder as the script.

3\. Run the script to see the analysis and visualizations.
